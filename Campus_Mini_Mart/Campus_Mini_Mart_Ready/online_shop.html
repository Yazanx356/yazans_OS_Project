Project Features

  • Responsive product grid for desktop and mobile screens.
  • Live search to find products quickly.
  • Category filtering for drinks, snacks, meals, and desserts.
  • Shopping cart saved using localStorage.

Project Goal

The goal of this website is to build a simple and user-friendly online store interface that demonstrates front-end development skills and basic cart functionality.

About the Store

About Campus Mini Mart

Campus Mini Mart is a sample e-commerce website designed for students who need a simple way to view products and manage purchases online. The website focuses on clarity, fast navigation, and an organized layout.

Technologies Used

Development Tools

  • HTML for page structure and content.
  • CSS for styling, layout, and responsive design.
  • JavaScript for cart logic, filtering, and search features.
  • localStorage to save cart items in the browser.
